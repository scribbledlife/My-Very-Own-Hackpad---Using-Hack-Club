# My Very Own Hackpad (6-Key Macropad)

**Built for Hack Club** üöÄ

This is my first ever build from scratch. This is a macropad with 6 keys, made by using the tutorial provided by Hack Club.

My name is **Mridul Goel**, and I am a sophomore. I aspire to be a future aerospace engineer, and as such I thought that it would be a good idea to get into this kind of stuff. What you are looking at is a 6-key macropad that I made for my first ever project, just to get the hang of things and different kinds of softwares that are available.

I mainly used **KiCad** for the schematic and the main PCB wiring. Then I moved onto **Fusion 360**, working on the bottom and top case to house the microcontroller and everything.

**NOTE**: I modified the original design by changing it from 4 keys to 6 keys, because I found out that this is supposed to be modified to be counted as your own‚Äîand well, who doesn't like some more keys right! üòé
